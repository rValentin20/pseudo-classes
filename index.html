Pseudo-classes

:active

Representa un elemento (como un botón) que el usuario está activando.

Activado

:cheked

El usuario puede activar este estado marcando/seleccionando un elemento, o desactivándolo desmarcando/deseleccionando el elemento.

:default

Representa a cualquier elemento deshabilitado. Un elemento se encuentra deshabilitado si no puede ser activado (por ejemplo ser selecionado, hacer click en él o aceptar texto de entrada) ni aceptar el foco.

:dir

Coincide con los elementos en función de la direccionalidad del texto contenido en ellos.

:disabled

Representa a cualquier elemento deshabilitado. Un elemento se encuentra deshabilitado si no puede ser activado (por ejemplo ser selecionado, hacer click en él o aceptar texto de entrada) ni aceptar el foco.

:empty

Representa cualquier elemento que no tenga hijos. Como este cuadro.

:enabled

Representa cualquier elemento habilitado. Un elemento está habilitado si puede ser activado (es decir seleccionado, se puede hacer click en él, acepta que se le introduzca texto, etc.) o si accepta el foco.


:first

    Pseudo-elementos

    ::after

    Añadir algún contenido despues, por medio de una indicación.

    ::before

    Añadir algún contenido antes, por medio de una indicación.

    ::first-letter

    Letra capital de un texto. También es modificable su suente y color.

    ::first-line

    Los estilos se mantendrán en la primeria línea sin importar los breakpoints.

    ::selection

    Al seleccionar con el puntero una parte del parrafo se marcará de un color seleccionado en estilos. Como cuando se copea un texto.

    ::backdrop

    Es un cuadro del tamaño de la ventana gráfica que se representa inmediatamente debajo de cualquier elemento que se presente en modo de pantalla completa.

    ::placeholder

    Texto provisional que puede dar indicaciones "" en un elemento como input donde es utilizado para introducir datos.
